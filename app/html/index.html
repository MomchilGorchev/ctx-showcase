Colorful circles with random size and position

Download source

Highlights

The source code for this example can be found on GitHub.
This example is using TweenMax's SlowMo easing (more info here) and the commonly used function to generate random array indexes: 

                    function randomNumber(min, max) {
                        return Math.random() * (max - min) + min;
                    }

The the duration of the animation of each object is based on the it's radius value. Smaller circles gets faster animation. Every shape/circle (however you want to call it :)) is initially being draw on a random canvas position. Then the x and y are being changed to another random value and the object is moved to its new position. 

                    function moveAround(dot){

                        var pos = {
                            x: dot.x,
                            y: dot.y
                        };
                        var newPos = {
                            x: Math.random() * canvas.width ,
                            y: Math.random() * canvas.height
                        };

                        var timing = dot.radius < 5 ? 10 : 30;

                        TweenMax.to(dot, timing, {
                            x: newPos.x,
                            y: newPos.y,
                            ease: SlowMo.ease.config(0.3, 0.4, false),
                            onComplete: function() {
                                TweenMax.to(dot, timing, {
                                    x: pos.x,
                                    y: pos.y,
                                    ease: SlowMo.ease.config(0.3, 0.4, false),
                                    onComplete: function() {
                                        moveAround(dot);
                                    }
                                })
                            }
                        });
                    }
